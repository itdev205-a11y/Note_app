# 📝 Note App

A modern and simple Flutter notes application that allows users to create, view, manage, and delete notes with a clean and responsive user interface.

---

## ✨ Features

- 📝 Create new notes
- 📋 Display notes list
- 🔍 View note details
- 🗑️ Delete notes
- ♻️ Restore deleted notes from Trash
- 🎨 Clean Material UI design
- 📱 Responsive layout
- ♻️ Reusable Widgets
- ⚡ Smooth user experience

---

## 🛠️ Technologies Used

- Flutter
- Dart
- Material Design
- StatefulWidget
- Custom Widgets
- Local State Management
